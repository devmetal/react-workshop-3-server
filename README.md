# React Workshop 3 Server

## Telepítés és futtatás

Miután lehúztad a repót, add ki az install parancsot.

Ezzel párhuzamosan szükséged lesz egy mongodb adatbázisra. Erre a célra a mongolabot javaslom.

Hozz létre egy .env fájlt a következő tartalommal:
```
MONGO_URI=eléeési út a db-hez
PORT=A port ahol szeretnéd futtatni a szervert
```

Ezután a start parancsal indítsd el a szervert. Első indításkor felseedeli az adatbázist.

Ezután a szerver használható lesz a workshop-3 klienssel.
